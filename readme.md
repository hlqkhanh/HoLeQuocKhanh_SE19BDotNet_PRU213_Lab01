## 📖 Project Overview
**Space Explorer** is an endless-runner arcade game where players navigate a high-speed spaceship through a dangerous asteroid field. The game focuses on reflex-based gameplay, resource management (score as health), and dynamic kinesthetic feedback.

### 🎮 Core Mechanics
* **Score-based Health System:** The player starts with **30 Points**.
  * 🌟 **Collect Stars:** +10 Points.
  * 🪨 **Hit Asteroids:** -30 Points.
  * 💀 **Game Over:** Triggered dynamically when the score drops to 0 or below.
* **Combat:** Players can shoot lasers to destroy incoming asteroids and clear a path.

---

## ✨ Key Features (Creativity & Polish)
- [x] **Dynamic Audio System:** Integrated Background Music (BGM) loop and high-quality Sound Effects (SFX) for explosions, pickups, and game over states.
- [x] **Game Feel (Camera Shake):** Custom Coroutine logic to trigger a randomized screen shake effect upon violent collisions (asteroid impacts), enhancing the visual feedback.
- [x] **Data Persistence:** Final scores are securely cached using `PlayerPrefs` and displayed on the End Game screen.
- [x] **Professional UI/UX:** Neon Sci-Fi aesthetic utilizing **TextMeshPro**, automated layouts, and interactive Mission Briefing panels.
